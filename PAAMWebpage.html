PAAM-GENERATION



Introduction

Machine learning (ML) over Phylogeny and Abundnace -aware matrix (PAAM) termed as PAAM-ML, is a supervised ML method for classifying metagenomic samples into Phenotype taking advantage of the evolutionary structure of microbial community as driven by phylogenetic trees along with quantitative abundances.PAAM is constructed with Perl based Scripting, which could be further used in ML.This page link guides the construction of a novel data structure of PAAM.

Downloads

The Perl source code is available here. Dependency : PERL Tk Module for GUI.

Instructions

Run with command on Commnd Prompt

>>perl PAAM_gen.pl

It provides a GUI to input : (i) Taxa/OTU Abundance Count Table and (ii) Phylogenetic Tree connecting taxa & Generate PAAM from these two Inputs

Input data files Needed and formats

  • OTU count data table
  • Phylogenetic tree

OTU count data Data .csv file that has OTU count information for each sample and phenotypic information in the last Column. It serves as table where rows denote the metagenomic samples and columns denote the Taxa/OTUs

An input sample file is found here

Phylogenetic tree

Data file that contains the phylogenetic tree for the Taxa in a Newick format.

A tree related to the sample OTU table is available at here

It needs to be preprocessed with following Shell command in Linux before is it inputted to the application of Perl for generating PAAM.

>> throat.tre |sed "s/(/(\n/g" |sed "s/)/\n)/g" |sed "s/,(/,\n(/g" > T3.tre

A preprocessed tree is shown at here

Use preprocessed tree (e.g. "T3.tre") and OTU Count Table ("THOTU.csv") in the Perl Script as shown here

Give path for output file to be generated../../PAAM.csv

Output Data File

Click Generate Button to create PAAM.csv.
