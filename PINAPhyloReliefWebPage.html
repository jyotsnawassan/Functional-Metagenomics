PINA-PhyoRelief



Introduction

PINA is a recently developed phylogenetic index to find the phylogenetic similarity between microbiome samples instead of the UniFrac ranking of features representing internal nodes in phylogenetic tree topology.

where Î¦ij represents cophenetic distance matrix or co-occurrence association matrix; t he entries in the matrix are the sum of the length of the branches separating each pair of microbial species. Thus, if two species are far apart on the phylogeny, their distance will be larger than that for two closely related species.; A and B represent microbial samples with nA and nB microbial species that have been sampled

Instructions

Run with command on Commnd Prompt

perl script name -OTUS_csv -ABUNDANCE_csv -phylogeny -filter < 0.0 - 1.0 > -OUT_W -LOG

Input data files Needed and formats

  • OTU PINA based relation File
  • OTU count data table
  • Phylogenetic tree
  • Filter

Pina based relation needs co-occurrence Matrix as Input Co-occurrence is calculated via R Scipt using ape package in R

Sample R Script is found here

OTU count data Data .csv file that has OTU count information for each sample and phenotypic information in the last Column. It serves as table where rows denote the metagenomic samples and columns denote the Taxa/OTUs

Phylogenetic tree Data file that contains the phylogenetic tree for the Taxa in a Newick format.

It needs to be preprocessed with following Shell command in Linux before is it inputted to the application of PINA PhyloRelief .

>> sample.tre |sed "s/(/(\n/g" |sed "s/)/\n)/g" |sed "s/,(/,\n(/g" > T3.tre

filter Used to filter ancestral nodes (e.g. 0.50 as filter indicates 50 % of teh ancestral nodes are to selected)

OUT_W Generates Output
